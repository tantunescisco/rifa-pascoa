⚙️ Configuração Firebase necessária

Para funcionar online, a rifa usa o Firebase Realtime Database.
Segue os passos:

  1. Vai a console.firebase.google.com e cria um projeto.
  2. Clica em Realtime Database → Criar base de dados → Modo de teste.
  3. Em Definições do projeto → As tuas apps, adiciona uma Web App e copia o objeto firebaseConfig.
  4. Abre o ficheiro rifa-pascoa.html e substitui os valores no bloco FIREBASE CONFIG no início do script.
